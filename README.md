# insult-generator

### An API that returns insults, compliments, or comebacks

#### Endpoints:
- /insult
- /compliment 
- /comeback

#### Running locally:
```
$ cd <path/to/inslt-generator>
$ make run
```


#### Info:
- Leveraged insults found here: https://www.randominsultgenerator.com/ to create this API.
- Designed to be run on Heroku
